#requisiti
-Installare nodejs 
-Installare cordova  comando:npm install -g cordova
-Installare electron comando:npm install electron --save-dev 
-Installare angular: npm install -g @angular/cli
-Installare mongodb

#Lancio DB
-Lanciare mongod sulla cartella di installazione esempio : "C:\MongoDB\Server\4.0\bin\mongod"

#Lancio Server
-Posizionarsi nella cartella server
-Scaricare moduli necessari con: npm install
-Lanciare App con node battaglia.js


#Lancio Applicazione Angular 
-Posizionarsi nella cartella client
-Scaricare moduli necessari con: npm install
-Lanciare App con ng serve --open


#Lancio electron
-Lanciare l'eseguibile del proprio sistema operativo
-In alternativa posizionarsi nella cartella electron e s caricare moduli necessari con: npm install
-Lanciare il comando npm run electron


#Cordova
-Lanciare emulatore android (ad esempio Android virtual device Manager 
-Entrare in cartella cordovabattle

-Lanciare comando:
cordova run android
